# NBS Offline V44

This is an offline client for V44 (the version that added Otis).

## Features

- Main menu
- Offline battles
- Colorfull lobby
- Disabled store and Brawl Pass
- Random bot names
- A lot of random stuff
- Edit controls brawler is now Otis
